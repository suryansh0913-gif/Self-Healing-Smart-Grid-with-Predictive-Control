# Self-Healing Smart Grid with Predictive Control

## Description
An ESP32-based smart grid prototype for real-time fault detection, automatic fault isolation, and backup power restoration using predictive threshold-based control.

## Features
- Real-time fault detection
- Automatic fault isolation
- Backup power restoration
- Predictive threshold-based control
- Relay-based autonomous switching

## Hardware Used
- ESP32
- ACS712 Current Sensor
- ZMPT101B Voltage Sensor
- Relay Module

## Programming Language
- Embedded C
